# RawTX
RawTX is a simple python script that helps you get the RSZ values of a RAW bitcoin transaction signature.
```
usage: rawtx.py [-h] [--rawtx RAWTX]

This tool helps to get ECDSA Signature RSZ values from Bitcoin rawtx!

optional arguments:
  -h, --help     show this help message and exit
  --rawtx RAWTX  Raw Transaction on the blockchain

Enjoy the program!
```
